# Deploy to Vercel (Share with Friends)

## Quick Steps:

1. **Go to Vercel:** https://vercel.com
2. **Sign up/Login** with your GitHub account
3. **Click "Add New Project"**
4. **Import your repository:**
   - Select `jameschhetree/hldashboard`
   - Click "Import"
5. **Configure (usually auto-detected):**
   - Framework Preset: Next.js (should auto-detect)
   - Root Directory: `./` (default)
   - Build Command: `npm run build` (default)
   - Output Directory: `.next` (default)
6. **Click "Deploy"**

That's it! Vercel will:
- Build your app
- Give you a live URL (like `hldashboard.vercel.app`)
- Auto-deploy when you push to GitHub

## After Deployment:

- You'll get a URL like: `https://hldashboard-xxxxx.vercel.app`
- Share this URL with friends!
- Every time you push to GitHub, it auto-updates
